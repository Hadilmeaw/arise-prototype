ARISE prototype

Two interactive web applications accompanying the D2.2 UI/UX Specifications. The Coach guides the patient through a Sit-to-Stand session with live skeleton feedback. The Therapist dashboard gives clinicians caseload, sessions, and replay tools.

Status Phase 1 prototype Stack React 19, Vite, MediaPipe Pose Persistence Browser localStorage (no back-end yet)
Patient-facing

Coach app

Full-screen calm surface with live skeleton overlay, automatic calibration, real-time corrective feedback, day-grouped history, and per-device settings. Built for the acquisition node in the clinical lab.

Clinician-facing

Therapist app

Desktop dashboard with caseload, per-patient charts, session replay with rep-level scrub bar, editable session notes, and longitudinal reports. Built for the clinician's browser.

This is a Phase 1 prototype. It uses fake fixture data and stores state in the browser only. No real patient information is involved. Production (WP4) replaces localStorage with the EU-region cloud back-end specified in D2.1B and the on-device acquisition pipeline running on a Jetson Orin with an RGB-D camera.
